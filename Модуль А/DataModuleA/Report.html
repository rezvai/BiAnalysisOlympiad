Описание атрибутов базы данных


Краткое описание значимых атрибутов базы данных

Название вакансии - данный атрибут дает понять по названию специфику вакансии.

описание вакансии - данный атрибут содержит в себе описание, что требуется от кандидата, а также, что готов дать за работа работодатель.

Заработная плата- данный атрибут является самым главным для сотрудника, который собирается устраиваться на работу.

Ключевые навыки - данный атрибут отражает самые важные навыки, которые неоьходимо иметь сотруднику.


Краткое описание всех атрибутов базы данных: заполненные и пропущенные значениея; плонтость распределения пропущенных значений; оценка погрешности

Формула плотности распределения данных: (кол-во пропущенных значений/кол-во всех значений) * 100%

Формула оценка погрешности при допустимом значение в 50%: пплотность рапсределения - 50%
